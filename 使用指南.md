# ZelosTech 论坛自动化使用指南

## 🚀 快速开始

### 方法一：双击运行（推荐）

1. **双击 `start_forum_automation.bat`**
   - 系统会自动检查 Python 环境
   - 安装必要的依赖包
   - 测试论坛连接

### 方法二：手动运行

#### 步骤 1：检查 Python 环境
```bash
python --version
```
确保显示 Python 3.8 或更高版本

#### 步骤 2：安装依赖包
```bash
python -m pip install requests beautifulsoup4
```

#### 步骤 3：测试连接
```bash
python simple_test.py
```

#### 步骤 4：运行自动化（如果测试成功）
```bash
python run_local.py
```

## 🔧 故障排除

### 问题 1：Background Agent 不可用
**原因**：Background Agent 是预览功能，可能还未对所有用户开放

**解决方案**：
- 使用本地运行方式
- 等待 Cursor 更新后再次尝试
- 检查 Cursor 设置中的 Beta 功能是否启用

### 问题 2：Python 环境问题
**原因**：系统中有多个 Python 版本，环境混乱

**解决方案**：
1. 卸载多余的 Python 版本
2. 重新安装 Python 3.8+
3. 确保添加到系统 PATH

### 问题 3：依赖包安装失败
**原因**：Python 版本过新，某些包不兼容

**解决方案**：
- 使用简化版依赖（已配置）
- 降级到 Python 3.8-3.11
- 使用虚拟环境
